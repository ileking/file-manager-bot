# Telegram File Manager Bot

Telegram-бот для хранения и управления файлами с поддержкой категорий, версий, комментариев, поиска, скачивания и удаления файлов.

## Технологии

- Python
- Telegram Bot API
- python-telegram-bot
- SQLite
- pathlib / os
- dotenv

## Возможности

- загрузка документов и фото;
- выбор категории файла;
- добавление комментария;
- автоматическое создание версий при повторной загрузке файла с тем же названием;
- просмотр списка файлов;
- поиск файлов;
- скачивание файла по ID;
- удаление файла по ID;
- изменение комментария;
- просмотр статистики хранилища;
- обработка ошибок и неизвестных команд.

## Установка

```bash
python -m venv .venv
```

Windows:

```bash
.venv\Scripts\activate
```

macOS / Linux:

```bash
source .venv/bin/activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

Создайте файл `.env` рядом с `main.py` и добавьте токен бота:

```env
BOT_TOKEN=ваш_токен_от_BotFather
```

## Запуск

```bash
python main.py
```

## Основные команды

- `/start` — запуск бота
- `/help` — справка
- `/upload` — инструкция по загрузке файла
- `/files` — список всех файлов
- `/categories` — список категорий
- `/search название` — поиск файла
- `/versions название_файла` — версии файла
- `/download ID` — скачать файл
- `/delete ID` — удалить файл
- `/comment ID текст` — изменить комментарий
- `/info ID` — информация о файле
- `/last` — последние файлы
- `/stats` — статистика
- `/cancel` — отмена действия

## Демонстрация работы

1. Пользователь запускает бота командой `/start`.
2. Пользователь отправляет файл.
3. Бот предлагает выбрать категорию.
4. Пользователь пишет комментарий.
5. Бот сохраняет файл и присваивает ему ID.
6. Пользователь смотрит список файлов через `/files`.
7. Пользователь скачивает файл через `/download ID`.
8. Пользователь загружает файл с тем же названием, и бот создаёт новую версию.
9. Пользователь смотрит версии через `/versions название_файла`.
10. Пользователь удаляет файл через `/delete ID`.
